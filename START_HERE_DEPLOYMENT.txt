DEPLOYMENT STEPS (QUICK VERSION)
================================================================================

1. Generate secret:
   openssl rand -hex 32

2. Get Resend API key:
   https://resend.com/api-keys

3. Deploy edge functions:
   - Go to Supabase Dashboard → Functions
   - Deploy run-health-checks (uncheck "Verify JWT")
   - Deploy send-health-alert (uncheck "Verify JWT")

4. Add secrets to Supabase Vault:
   - HEALTHCHECK_SECRET (from step 1)
   - RESEND_API_KEY (from step 2)

5. Run database migration:
   - Go to Supabase SQL Editor
   - Copy/paste DEPLOYMENT_MIGRATION_WITH_ROLLBACK.sql
   - Run

6. Verify:
   SELECT * FROM health_checks ORDER BY created_at DESC LIMIT 5;
